HTTP и API

API (Application Programming Interface) - программный интерфейс приложения. Интерес для общения между положениями.
GUI (Graphic User Interface) - графический интерфейс пользователя. 
CLI (Command Line Interface) - интерфейс командной строки. Когда обращаешься к программе с помощью командной строки.

Протокол - договоренность, как отсылать и получать информацию, и как ее понимать.
В протоколе важно: 
1. Способ передачи (костер, радио-волны, с помощью чего передаем)
2. Формат сигналов пришедших с помощью выбранного способа (огонь, дым)

Протокол HTTP - протокол прикладного уровня, т.к. как он предназначен только для передачи текстовых данных в определенном формате.

API
- Способ общения одной программы с другой
- Например, клиента с сервером
- Клиент создает HTTP-запросы
- Сервер их обрабатывает и возвращает ответ
- Клиент может реагировать на ответ сервера
